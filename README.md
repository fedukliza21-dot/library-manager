# 📖 Library Manager

Консольное приложение на Python для управления личной библиотекой.

Позволяет добавлять, удалять, искать и редактировать книги.  
Данные сохраняются в JSON-файл.

---

## 📌 Возможности

- Добавление книги
- Поиск по автору
- Удаление книги (по id или названию)
- Изменение статуса и рейтинга
- Сохранение данных в файл

---

## ⛓ Технологии

- Python 3
- JSON
- CLI (консольное приложение)

---

## ⏱ Быстрый запуск (1 минута)

### 1. Склонировать репозиторий

```bash
  git clone https://github.com/fedukliza21-dot/library-manager.git
```
### 2. Перейти в папку проекта
```bash
  cd library-manager
```
### 3. Создать файл для хранения данных
Создайте файл books.json и добавьте в него:
```
[]
```
### 4. Запустить программу
```bash
  python main.py
```
### Пример данных
```
[
    {
        "id": 1,
        "name": "1984",
        "author": "Оруэлл",
        "genre": "Антиутопия",
        "year": 1945,
        "status": "Прочитана",
        "rating": "5"
    }
]
```
### 💡 Примечания
```
1. Если файл books.json отсутствует, его нужно создать вручную
2. Все изменения сохраняются между запусками программы
3. Поиск выполняется без учёта регистра
```
